## ADDED Requirements

### Requirement: 하루 뷰에 24시간 타임라인이 표시된다
시스템은 선택된 날짜의 이벤트를 00:00~24:00 세로 타임라인에 시각적으로 표시해야 한다.

#### Scenario: 당일 이벤트 타임라인 표시
- **WHEN** 사용자가 하루 뷰를 열면
- **THEN** 해당 날짜에 등록된 이벤트가 시작~종료 시간 위치에 블록으로 표시된다

#### Scenario: 이벤트 없는 날
- **WHEN** 해당 날짜에 이벤트가 없으면
- **THEN** 빈 타임라인이 표시되고 일정 추가를 유도하는 버튼이 보인다

#### Scenario: 날짜 이동 시 타임라인 갱신
- **WHEN** 사용자가 DayNavigator로 다른 날짜로 이동하면
- **THEN** 해당 날짜의 이벤트만 타임라인에 표시된다
