# Product Brief

## 1. Product Summary

이 서비스는 **대학생**이 **수업, 과제, 일정이 뒤섞여 관리하기 어려운 문제**를 **하루 할 일 + 월간 일정 + 과제 마감 자동 반영이 통합된 스터디 플래너**로 해결하도록 돕는다.

---

## 2. Problem

대학생은 강의, 과제, 시험, 개인 일정을 동시에 관리해야 하지만 각각 다른 앱이나 노트에 흩어져 있어 빠뜨리는 경우가 많다. 과제 마감일을 알고 있어도 그날이 오기 전까지 매일 "오늘 해야 할 일" 목록에 직접 옮겨 적어야 하는 번거로움이 있다. 할 일을 짧게 메모하면 나중에 context를 잊어버리고, 길게 적으면 목록이 복잡해져 오히려 보기 불편하다. 플래너를 별도 앱으로 열어야 한다면 쉽게 잊고 사용 빈도가 떨어진다. 결국 아날로그 다이어리로 돌아가거나, 아무것도 쓰지 않게 된다.

---

## 3. Target User

- **주요 사용자:** 과제·시험·개인 일정을 함께 관리해야 하는 대학생
- **사용 상황:** 하루를 시작하기 전 오늘 할 일을 확인하거나, 수업 후 과제를 등록할 때. 바탕화면에 항상 띄워두고 틈틈이 확인
- **사용자의 기술 수준:** 스마트폰과 PC에 익숙하지만 복잡한 설정 없이 바로 쓰고 싶은 일반 사용자
- **사용자가 기대하는 결과:** 오늘 뭘 해야 하는지 한눈에 보이고, 과제 마감을 놓치지 않으며, 할 일을 체크했을 때 성취감을 느끼는 것

---

## 4. Core Value

과제 마감일을 한 번만 입력하면 마감 당일까지 매일 할 일 목록에 자동으로 나타나, 대학생이 아무것도 빠뜨리지 않고 하루를 계획할 수 있다.

---

## 5. MVP Scope

이번 MVP에 포함할 것:

- **하루 뷰(Daily View):** 날짜별 할 일 목록, 체크박스로 완료 표시, 클릭 시 상세 내용 모달 표시
- **월간 뷰(Monthly View):** 달력 형태로 일정·과제 마감일 표시, 날짜 클릭 시 해당 날의 할 일로 이동
- **과제/일정 등록:** 과목명 + 내용 + 마감일 입력 시, 마감일까지 매일 할 일 목록에 자동 반영
- **바탕화면 상주(Always-on-top 위젯):** 창을 닫지 않아도 항상 바탕화면에 떠 있는 형태 (Electron 또는 브라우저 고정 탭)
- **로컬 데이터 저장:** 로그인 없이 브라우저 localStorage에 데이터 유지

---

## 6. Non-goals

이번 MVP에서 제외할 것:

- 로그인 / 회원가입
- 결제
- 실시간 협업 (공유 플래너)
- 대용량 파일 업로드
- 다중 외부 API 연동 (Google Calendar 동기화 등)
- 복잡한 관리자 권한
- 모바일 앱 (웹 우선)
- 알림/푸시 기능 (MVP 이후 고려)

---

## 7. Success Criteria

- 사용자는 핵심 기능(할 일 추가 → 체크 완료)을 **1분 안에** 사용할 수 있다.
- 3회차 종료 시 **하루 뷰 + 월간 뷰 + 과제 자동 반영 흐름**을 브라우저에서 확인할 수 있다.
- 4회차에서 **과제 등록 → 마감일까지 할 일 자동 생성 → 체크 완료** 핵심 흐름을 Playwright 테스트로 검증할 수 있다.
- 최종 결과물은 **Vercel 등에 배포 가능한 형태**여야 한다.
