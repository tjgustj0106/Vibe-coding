# 잔소리 캐릭터 기능 계획서

## 1. 개요

사용자가 이름·사진·성격을 등록하면, 등록한 캐릭터가 마감 임박 시 잔소리를 하거나
할 일 완료 시 칭찬을 해주는 동기부여 기능.

**완전 무료** — API 없이 프리셋 대사 뱅크로 구현.

---

## 2. 온보딩 플로우

앱 최초 진입 시 세팅 화면 표시 (이후 설정에서 변경 가능).

```
Step 1. 이름 입력
  → "누가 응원해줬으면 하나요? 이름을 입력해주세요."
  → 예: 엄마 / 친구 이름 / 멘토 이름

Step 2. 사진 업로드 (선택)
  → 파일 선택 or 건너뛰기
  → base64로 localStorage 저장 (최대 500KB 권장)

Step 3. 성격 선택
  → 5가지 카드 중 1개 선택
  → 각 카드에 대표 대사 미리보기 표시

Step 4. 완료
  → "오늘부터 [이름]이/가 응원할게요!"
```

---

## 3. 표시 위치 & 트리거 조건

### 3-1. 하루 뷰 상단 — 마감 재촉

| 조건 | 표시 여부 |
|---|---|
| 미완료 할 일 중 마감 D-3 이하인 것이 1개 이상 | ✅ 표시 |
| 미완료 할 일이 없거나 모두 D-4 이상 | ❌ 숨김 |

- 가장 급한 과제 1개를 기준으로 대사 선택
- 사진(아바타) + 말풍선 형태

### 3-2. 완료 리액션 — 체크 시 잠깐 등장

- 할 일 체크 버튼 클릭 → 아바타 + 칭찬 말풍선 2초 표시 후 사라짐

### 3-3. 주간 진도 코멘트 — 하루 뷰 하단

- 이번 주(월~일) 할 일 완료율 계산
- 완료율 80% 이상 → 긍정 코멘트
- 완료율 30% 미만 → 부정 코멘트
- 그 사이 → 표시 안 함

## 5. 데이터 모델

```ts
// localStorage 키: "studylog_coach"
type CoachProfile = {
  name: string;               // 표시 이름
  photo: string | null;       // base64 이미지 or null (기본 이모지)
  personality: PersonalityType;
};

type PersonalityType =
  | "warm"       // 다정한
  | "strict"     // 엄하고 무서운
  | "friend"     // 친한 친구
  | "tsundere"   // 츤데레
  | "blunt";     // 무뚝뚝한
```

## 6. 컴포넌트 구조

```
src/features/coach/
├── types.ts                   # CoachProfile, PersonalityType
├── storage.ts                 # getCoach / saveCoach
├── dialogue.ts                # 성격별 대사 뱅크 + getDialogue(personality, situation, taskTitle?)
├── hooks/
│   └── useCoach.ts            # 프로필 로드/저장 훅
└── components/
    ├── CoachOnboarding.tsx    # 최초 설정 화면 (이름·사진·성격 선택)
    ├── CoachBubble.tsx        # 아바타 + 말풍선 UI
    ├── CoachCompletionPop.tsx # 완료 시 팝업 (2초 후 자동 닫힘)
    └── CoachWeeklySummary.tsx # 주간 진도 코멘트
```

---

## 7. 구현 순서

1. `dialogue.ts` — 대사 뱅크 작성 및 `getDialogue()` 함수
2. `storage.ts` + `types.ts` — 데이터 모델
3. `useCoach.ts` — 프로필 훅
4. `CoachOnboarding.tsx` — 최초 설정 화면
5. `CoachBubble.tsx` — 아바타 + 말풍선 기본 UI
6. `page.tsx` 연결 — 하루 뷰 상단에 마감 재촉 표시
7. `CoachCompletionPop.tsx` — 체크 완료 시 팝업
8. `CoachWeeklySummary.tsx` — 주간 진도 코멘트
